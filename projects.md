# Enterprise Network Infrastructure Design

This project demonstrates the design and implementation of a multi-building enterprise network using **Cisco Packet Tracer**. The network includes routing, VLAN segmentation, DHCP, wireless connectivity, VoIP, and centralized services across two buildings. :contentReference[oaicite:0]{index=0}

## Technologies Used

- Cisco Packet Tracer
- Routing
- VLANs
- DHCP
- Static Routing
- Wireless Access Points
- VoIP
- Network Design
- IP Addressing & Subnetting
